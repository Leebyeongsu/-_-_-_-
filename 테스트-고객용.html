고객용 접속 테스트

아래 링크를 클릭하여 고객용 화면이 정상적으로 표시되는지 확인하세요:

  • 🔧 관리자용 (QR 생성 버튼 표시)
  • 👤 고객용 (QR 생성 버튼 숨김)
  • 📝 고객용 (apply 파라미터)

기대되는 결과:

관리자용:

  • ✅ "📱 QR 코드 생성" 버튼 표시
  • ✅ "카카오톡 공유" 버튼 표시
  • ✅ 제목: "구포현대아파트 통신 환경 개선 신청서"

고객용:

  • ❌ "📱 QR 코드 생성" 버튼 숨김
  • ❌ "카카오톡 공유" 버튼 숨김
  • ✅ "신청서 제출" 버튼만 표시
  • ✅ 제목: "통신 환경 개선 신청서"
  • ✅ 신청 완료시: 신청번호, 공사요청, 연락처, 통신사, 희망시작일, 요청사항, 신청일시 표시
  • ✅ 완료 버튼: "✅ 신청 완료" (창 닫기)
  • ✅ 담당자 연락 안내 메시지 표시

🧪 테스트 순서:

  1. 고객용 링크 클릭
  2. 간단한 정보 입력 후 "신청서 제출" 클릭
  3. 신청 완료 화면에서 신청번호, 공사요청, 연락처, 통신사, 희망시작일, 요청사항, 신청일시가 표시되는지 확인
  4. "✅ 신청 완료" 버튼이 표시되는지 확인
  5. 버튼 클릭시 완료 메시지와 창 닫기 기능 확인
